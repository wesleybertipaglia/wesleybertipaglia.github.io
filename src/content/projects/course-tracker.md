## Course Tracker

**Course Tracker** is an innovative platform designed to help users organize and track their learning progress through online courses. With its intuitive interface, users can:

- **Create Courses**: Easily set up new courses to structure their learning.
- **Add Lessons**: Incorporate YouTube videos as lessons, making it simple to access instructional content.
- **Enroll in Courses**: Join courses that interest you and follow along at your own pace.
- **Track Progress**: Mark lessons as completed to keep track of your learning journey.

With Course Tracker, mastering new skills and knowledge has never been easier or more organized!

### Tech Stack

- **Python**: The backend is built using Python, ensuring a robust and scalable application.
- **Flask**: This lightweight web framework allows for rapid development and deployment.
- **SQLAlchemy**: An ORM that simplifies database interactions and management.
