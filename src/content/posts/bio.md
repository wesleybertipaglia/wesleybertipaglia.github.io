# Bio Website

Bio is a simple website project that allows you to create and manage a list of links. It provides a user-friendly interface to store and organize your favorite URLs in one central place. It's inspired in Linktree and some popular Instagram bio websites.

The Bio is built with React, TailwindCSS, Phosphor icons and Jest for testing.

## Technologies

- React
- TailwindCss / PostCss
- Phosphor icons
- Jest
- ESLint
